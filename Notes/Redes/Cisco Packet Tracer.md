# Cisco Packet Tracer

- **Introdução**
    - Guias
        - Guia física: fornece uma interface par interagir com o dispositivo, incluindo ligá-lo, desligá-lo ou instalar módulos diferentes, como uma placa de interface de rede sem fio (NIC)
        - Guia Config: para dispositivos intermediários, como roteadores e switches, há duas maneiras de acessar as configurações do dispositivo. A GUI e a CLI.
        - Guia CLI: acesso a interface de linha de comando de um dispositivo cisco, requer conhecimento da configuração de dispositivos, habilidade necessária para implementação de rede  mais avançada.
        - Guia de Serviço:  um servidor tem todas as funções de um host com a adição de mais uma guia, permite que ele seja configurado com processos comuns de servidor, como HTTP, DHCP, DNS e outros
    - O Cisco tem um modo de aprendizagem
