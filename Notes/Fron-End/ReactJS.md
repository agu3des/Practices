### Anotações Curso

Biblioteca Js para criação de interfaces

Utilizado para construir SPAs (single page application)

Baseado em componentes

Utiliza JSX para renderizar HTML

Aplica virtual DOM para realizar as alterações DOM

Podemos adicionar a um projeto ou criar um projeto com ele

```jsx
create-next-app
```

JSX é como um HTML dentro do código JavaScript, é a principal maneira de escrever html em react

Pode-se interpolar variáveis, inserindo ela entre {}, sendo possível executar funções em jsx
